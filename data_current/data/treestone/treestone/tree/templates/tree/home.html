Welcome to the Heritage Structures Lab's Roman Materials Database! The Historic Materials Database is an interactive database of construction materials used in antiquity. This is the first step of a larger project that will enable a wider audience to create 3D and virtual reality (VR) models of historic structures. By helping a user build accurate models, the project will provide new perspectives of historic buildings and cities to historians, archaeologists, and others who study the past. The project offers an interactive map which can be used to gather information on construction materials within a geographical area based on the user's filters.
